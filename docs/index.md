## 未来展望

YIUI 将逐步接入 AI 功能，致力于实现 UI 开发的全自动化：

- **AI 友好文档** - 文档采用 Markdown 格式，便于 AI 识别和理解
- **YIUI 专属 MCP** - 开发中的 MCP 服务，支持在 Unity 中通过 AI 直接拼装 UI
- **全自动接管** - 最终目标：AI 全自动完成 UI 开发流程
