亦亦
ET9中推荐用自己包进行开发
每个人都需要做自己的包 如果每个人都手动其实要求还是蛮高的
这里提供一键生成  只需要给个名字 然后提供一些选项 最后就自动生成 文件夹程序集等各种操作
默认生成的包版本 = 0.0.0
0的版本可跳过包管理的其他功能 如更新检查 出现在版本管理等等...
0版本为本地私人使用版本 不影响任何包功能
如果要打包发布时请升级版本
