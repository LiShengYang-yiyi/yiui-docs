# YIUI 扩展包 AI 指导文档创建指南

> 本文档记录如何为 YIUI 扩展包创建 AI 指导文档

## 概述

为 YIUI 的每个扩展包创建详细的 AI 指导文档，方便开发者向 AI 助手咨询问题时获得准确的答案。

## 创建流程

### 1. 确定包映射关系

首先确定扩展包目录与 `cn.etetet.yiui*` 包的对应关系：

| 扩展包目录 | 对应包 |
|-----------|--------|
| audio-声音 | cn.etetet.yiuiaudio |
| condition-条件系统 | cn.etetet.yiuicondition |
| damagetips-伤害提示 | cn.etetet.yiuidamagetips |
| gameobjectpool-游戏对象缓存池 | cn.etetet.yiuigameobjectpool |
| luban-配置解决方案 | cn.etetet.yiuiluban |
| numeric-数值系统 | cn.etetet.yiuinumeric |
| superscroll-循环列表 | cn.etetet.yiuisuperscroll |
| 多语言-pro | cn.etetet.yiuilocalizationpro |

### 2. 创建目录结构

在每个扩展包目录下创建 `AI-指导/` 文件夹：

```
扩展包目录/
├── AI-指导/
│   ├── index.md           # 目录和快速问答
│   ├── 01-入门.md         # 系统架构、快速开始
│   ├── 02-配置.md         # 配置相关
│   ├── 03-代码使用.md     # 代码使用
│   ├── 04-xxx.md         # 根据复杂度添加更多
│   └── ...
```

#### 3.3 03-代码使用.md（核心）

**重要**：根据包的复杂度，拆分成多个文件：

```
AI-指导/
├── 03-代码使用.md        # Invoke 基础（简单介绍）
├── 04-功能A.md          # 功能A详细介绍
├── 05-功能B.md          # 功能B详细介绍
├── 06-功能C.md          # 功能C详细介绍
└── ...
```

每个文件只介绍一个功能板块，格式如下：

```markdown
# XXX - 功能名

> 本文件介绍 XXX 功能的详细使用方法

---

## 功能点1

### 方法

```csharp
// 方法签名
void DoSomething(param1, param2);
```

### 说明

- 能做什么1
- 能做什么2
```

**要点**：
1. 只说"能做什么"，不说"为什么"
2. 代码简略，有方法签名就行
3. 简洁明了
4. 不要解释原理，AI 会自己去了解

## 注意事项

1. **路径问题**：文档中的相对路径要确保正确
2. **链接检查**：VitePress 会检查死链接，确保链接指向的文件存在
3. **编码问题**：文件名使用中文时注意编码
4. **图片路径**：图片放在 `public/images/` 目录

## VitePress 配置

如果死链接较多，可在 `config.mts` 中添加：

```typescript
export default defineConfig({
  ignoreDeadLinks: true
})
```
