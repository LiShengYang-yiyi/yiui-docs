# 常量

 
![Image](/images/CB5ewIAQZiBf1Ykp2DDcMj22nHb_0_81cbc665.png)
  目前的目标是尽可能的把可能修改的一些参数提到常量中 
 这样可以最大程度的避免自己本地修改一些数据后更新被覆盖的问题 
 
 如: 
 修改了Root的分辨率  每个项目都有可能不一样  大小不一样 方向也不一样 
 YIUI默认提供的参数是 1920X1080 目前遇到反馈都需要修改这个 
 不过一更新包 就被覆盖了 有时候又忘记改回来导致一些BUG 浪费时间 
 
 所以现在提供了一个YIUIConst 配置 
 这个配置会生成在包外 由你自己管理 这样更新包也不会有什么问题了 
 
 不过也只能尽可能的避免 不能做到100% 
 如果你在使用中有什么觉得可以通过常量来解决的欢迎反馈 
 
 新项目工程 打开一次 YIUI工具>>常量 即可初始化
