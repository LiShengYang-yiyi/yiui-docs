亦亦
UI复杂结构下 特别结构又特别深又不知道ROOT在哪里时 找一个想要的目标是相对复杂的
现在只需点一点就可以直接跳转到对象上选中 大大提升体验
•
在场景视图里给每个 UI 元素（Image、Text 等）标注一个可点击的小按钮，让你直接点到它，不用在层级面板里慢慢找。
•
当鼠标靠近这些按钮时，会显示元素名字；如果附近有多个重叠元素，会弹出一个编号列表，让你快速识别是谁是谁。
•
解决 UI 层级复杂、元素重叠时“选不中”“选错层”的痛点，大幅减少在层级面板和场景里来回切换、错选的时间。
