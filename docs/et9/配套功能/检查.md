# 检查

 
![Image](/images/CB5ewIAQZiBf1Ykp2DDcMj22nHb_0_81cbc665.png)
 用户7857 UI 预制 脚本 一致性检查工具 
 随着项目的变大 
 各种需求的反复横跳 
 UI已经出现了冗余情况 
 以前写了一堆功能  现在要删掉 
 但是不太方便不好查找 手动删除又太慢 
 提供一系列简单的检查快捷方式 
 
 工具 > 检查 
 入口 
 
 
 如果检查到 有脚本 但是 没有预制 
 提供直接删除生成脚本 
 不包含非生成的脚本 
 
![Image](/images/GOBCwA1rdi7nBhkaBh8cLCjRnah_1_4483711c.png)
 
 如果有被删除的预制体 
 但是他还有脚本存在 
 则会出现删除按钮 
 根据需求可以一个一个删除 也可以使用 删除所有 
 删除后如果文件夹空了也会对应把文件夹删除 
 非YIUI创建的脚本无法删除  也不会出直接删除脚本文件夹功能 
 不建议在YIUI生成的文件夹下 创建自己的脚步 你应该自己独立管理 
 注意 这里需要你自行判断这个脚本是否可以删除 删除后有报错自行解决 
 
 
 如果检查到 有预制 但是 没有脚本 
 提供直接删除预制 
 不包含预制相关的其他文件夹 比如源文件 精灵图片...
