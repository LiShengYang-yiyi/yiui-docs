# GameObjectPool 游戏对象缓存池 - 入门指南

## 包信息

- **对应包**: `cn.etetet.yiuigameobjectpool`
- **依赖**: Luban 配置解决方案

## 系统架构

### 核心组件

**YIUIGameObjectPool** - 单例全局对象池管理器

### 核心功能

- 对象获取 (Get)
- 对象回收 (Recycle)
- 批量操作

## 快速开始

### 步骤 1: 安装

从 .Template 文件夹复制 config 包到 Packages

### 步骤 2: 配置

在 Luban 中配置对象池表

### 步骤 3: 使用

```csharp
// 获取对象
var obj = await YIUIGameObjectPool.Instance.Get(prefabPath);

// 回收对象
YIUIGameObjectPool.Instance.Recycle(obj);
```
