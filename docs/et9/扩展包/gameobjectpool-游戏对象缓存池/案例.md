 使用方法 
 
 删除原有的特效 使用空节点代替  挂载脚本  对象池触发器 
 填写对象名称 
 则 这个空对象 被激活时就会加载这个对象到自己的下面 
 自己被隐藏时  对象就会 回收到 对象池 
 自己被删除时 同样的也是 回收到对象池 
 
![Image](/images/AMMSwTy74ilXlukAHtucSBRYn0b_3_b02849be.png)
 同理 上面的选中确定时的特效也是这样操作的 
 配合YIUI显影操作更好用 
 
 实操 
 可以看到使用对象池过后 其他的预制体下是没有特效的 
 来回切换的时候才会有特效 这个特效就是复用的
