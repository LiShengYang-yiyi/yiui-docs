## 2. 文本获取

### 根据 Key 获取

```csharp
// 获取文本
var text = LocalizationHelper.GetText(key);
```

### 带参数

```csharp
// 参数替换 {0}, {1}
var text = LocalizationHelper.GetText(key, param1, param2);
```

### 默认值

```csharp
// key 不存在时返回默认值
var text = LocalizationHelper.GetText(key, defaultValue);
```

## 3. 语言切换

### 切换语言

```csharp
// 切换到指定语言
LocalizationManager.Instance.ChangeLanguage("zh-CN");
LocalizationManager.Instance.ChangeLanguage("en-US");
```

### 获取当前语言

```csharp
// 获取当前语言
var currentLang = LocalizationManager.Instance.CurrentLanguage;
```

### 获取可用语言

```csharp
// 获取所有可用语言
var languages = LocalizationManager.Instance.GetAvailableLanguages();
```

---

## 4. 参数替换

### 数字参数

```csharp
// 数字参数
LocalizationHelper.GetText("level_up", 10);
// "升级到 10 级"
```

### 名称参数

```csharp
// 名称参数
LocalizationHelper.GetText("get_item", itemName);
// "获得 {itemName}"
```

### 多个参数

```csharp
// 多个参数
LocalizationHelper.GetText("battle_result", winCount, totalCount);
// "战斗胜利 {0}/{1}"
```

## 5. UI 集成

### 静态绑定

```csharp
// 预制体上绑定
TextKeyComponent.SetKey("ui_login_title");
```

### 动态设置

```csharp
// 代码设置
textComponent.SetLocalizationKey(key);
```

### 刷新当前 UI

```csharp
// 刷新所有 UI 文本
LocalizationManager.Instance.RefreshAllUI();
```

## 6. 数据检查

### 检查 Key 存在

```csharp
// 检查 key 是否存在
var exists = LocalizationHelper.Exists(key);
```

### 检查所有 Key

```csharp
// 检查缺失的 key
var missingKeys = LocalizationHelper.GetMissingKeys(language);
```

### 导出报告

```csharp
// 导出检查报告
LocalizationHelper.ExportReport(language);
```
