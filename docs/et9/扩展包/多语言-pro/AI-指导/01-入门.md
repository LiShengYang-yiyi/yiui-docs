# 多语言 Pro - 入门指南

## 包信息

- **对应包**: `cn.etetet.yiuilocalizationpro`
- **依赖**: ET9 YIUI 框架、Luban 配置解决方案

## 系统架构

### 核心概念

1. **LocalizationHelper** - 多语言帮助类
2. **LocalizationManager** - 多语言管理器
3. **多语言表** - 存储多语言数据

### 功能特性

- Excel 数据源
- 数据检查
- 一键生成
- 按需加载

## 快速开始

### 步骤 1: 安装

安装 cn.etetet.yiuilocalizationpro

### 步骤 2: 配置

1. 创建 Excel 多语言表
2. 配置 Luban 导出

### 步骤 3: 使用

```csharp
// 获取多语言文本
var text = LocalizationHelper.GetText(key);

// 切换语言
LocalizationManager.Instance.ChangeLanguage(languageCode);
```
