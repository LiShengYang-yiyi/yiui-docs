 Model 可使用 
 Model层可直接访问到多语言信息

 实际开发时会发现Model层也是有可能需要的 可以让代码更直观方便 
 
 多语言生成注释 
 只有多有Key 还不够直观 
 在review 时查看key就可以直接知道多语言内容 
 提前判断使用方式,使用是否正确等等 
 
 自动补齐 
 在开发阶段可能有多种语言但是开发时都是用一种语言 
 策划在填写翻译时可能其他语言没有填写 
 但是程序方面如果切换语言会导致错误 等等问题 
 提供生成时自动补齐功能, 如果其他语言没有内容则使用 指定的语言/存在的 自动补齐 
 
 更方便的调用 
 不管是预制体上的静态绑定多语言 
 还是代码的动态设置多语言都提供简单方便的一键式设值 
 提供更多可能性扩展
