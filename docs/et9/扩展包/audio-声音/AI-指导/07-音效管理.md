## 7. 音效管理

### UI 音效

```csharp
// 点击音效
PlayUISound("click");

// 获得道具
PlayUISound("get_item");

// 升级
PlayUISound("level_up");

// 打开界面
PlayUISound("open_panel");

// 关闭界面
PlayUISound("close_panel");
```

### 战斗音效

```csharp
// 攻击
PlaySFX("attack");

// 暴击
PlaySFX("crit");

// 技能释放
PlaySFX("skill_cast");

// 命中
PlaySFX("hit");

// 闪避
PlaySFX("dodge");
```

### 播放控制

```csharp
// 停止所有音效
StopAllSFX();

// 暂停所有音效
PauseAllSFX();

// 恢复所有音效
ResumeAllSFX();
```

### 音效池

```csharp
// 复用 AudioSource
var source = SFXPool.Get();
source.Play();
SFXPool.Return(source);
```
