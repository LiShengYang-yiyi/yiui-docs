## 4. 音频配置

### 获取配置

```csharp
// 根据名称获取配置
var config = YIUIInvoke.GetAudioConfig("config_name");

// 获取音量
var volume = config.GetVolume();

// 获取音频名称列表
var names = config.GetAudioClipNames();

// 获取播放数量限制
var count = config.GetEmitterCount();
```

### 配置属性

```csharp
// 空间混合
config.GetSpatialBlend();

// 衰减距离
config.GetDistanceMin();
config.GetDistanceMax();

// 播放间隔
config.GetIntervalTimeMin();
config.GetIntervalTimeMax();
```

## 5. 播放控制

### 播放状态

```csharp
// 检查是否播放中
if (audioSource.isPlaying) { }

// 播放
audioSource.Play();

// 暂停
audioSource.Pause();

// 停止
audioSource.Stop();

// 继续
audioSource.UnPause();
```

### 播放位置

```csharp
// 获取播放位置
var time = audioSource.time;

// 设置播放位置
audioSource.time = 5.0f;

// 播放进度
var progress = audioSource.time / audioSource.clip.length;
```

### 循环

```csharp
// 开启循环
audioSource.loop = true;

// 关闭循环
audioSource.loop = false;
```
