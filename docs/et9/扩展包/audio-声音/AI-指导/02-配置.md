# Audio 声音 - 配置指南

## Audio 表配置

### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| Id | int | 配置 ID |
| SoundKey | string | 音频分组 Key |
| AudioClipNames | string | 音频资源名称，多个用逗号分隔 |
| Volume | float | 音量 (0-1) |
| EmitterCount | int | 同时播放数量 |
| SpatialBlend | float | 空间混合比例 (0=2D, 1=3D) |
| DistanceMin | float | 开始衰减距离 |
| DistanceMax | float | 完全衰减距离 |
| IntervalTimeMin | float | 最小播放间隔 |
| IntervalTimeMax | float | 最大播放间隔 |

### 配置示例

```
| Id | SoundKey | AudioClipNames | Volume | EmitterCount | SpatialBlend |
|----|----------|----------------|--------|--------------|---------------|
| 1  | BGM_Main | main_bgm.mp3   | 0.8    | 1            | 0             |
| 2  | SFX_UI   | click.mp3      | 1.0    | 5            | 0             |
| 3  | SFX_3D   | attack.mp3     | 0.6    | 10           | 1             |
```

## 进阶配置

### 配置枚举自动生成

利用配置枚举自动生成功能，可以：
1. 避免手动输入错误
2. 下拉选择音频
3. 预览试听

操作步骤：
1. 启用配置枚举自动生成
2. 在 Audio 表中添加特性
3. 重新导出配置

详见：[配置枚举自动生成](../配置枚举自动生成.md)

### YooAsset 组配置

在 YooAsset 中创建音频组：

1. 打开 YooAsset 工具窗口
2. 创建新的 AssetCollection
3. 添加音频文件（.mp3, .wav, .ogg）
4. 设置加载方式（Standalone, Streaming, Remote）

## 常见配置问题

### Q: 音频不播放怎么办？

检查清单：
1. ✅ Audio 表是否正确导出
2. ✅ YooAsset 资源是否添加到组
3. ✅ 资源路径是否正确
4. ✅ 音量是否设置为 0

### Q: 3D 音效没有空间感？

检查：
1. SpatialBlend 是否设置为 > 0
2. 音源与监听器距离是否在 DistanceMin/Max 范围内
3. 是否有其他 AudioSource 干扰

### Q: 如何实现音效防重叠？

使用 IntervalTimeMin/Max：
```csharp
var config = YIUIInvoke.GetAudioConfig("attack");
config.SetIntervalTimeMin(0.1f);  // 最小间隔 100ms
config.SetIntervalTimeMax(0.5f);  // 最大间隔 500ms
```
