## 9. 音量控制

### 全局音量

```csharp
// BGM 音量
AudioManager.SetBGMVolume(0.8f);

// 音效音量
AudioManager.SetSFXVolume(1.0f);

// 语音音量
AudioManager.SetVoiceVolume(1.0f);
```

### 静音控制

```csharp
// 静音
AudioManager.Mute(true);

// 取消静音
AudioManager.Mute(false);

// 检查静音状态
var isMuted = AudioManager.IsMuted;
```

### 渐变音量

```csharp
// 渐变到目标音量
AudioSourceFadeHelper.FadeTo(source, targetVolume, duration);

// 渐变 BGM
AudioManager.FadeBGMTo(0f, 2f);

// 渐变音效
AudioManager.FadeSFXTo(0.5f, 0.5f);
```

### 独立控制

```csharp
// 控制单个 AudioSource
audioSource.volume = 0.5f;

// 获取音量
var volume = audioSource.volume;
```

### 保存设置

```csharp
// 保存音量设置
AudioManager.SaveSettings();

// 加载音量设置
AudioManager.LoadSettings();
```
