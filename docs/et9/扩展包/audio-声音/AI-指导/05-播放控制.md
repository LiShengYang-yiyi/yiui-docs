## 5. 播放控制

### 播放 AudioClip

```csharp
// 直接播放
audioSource.clip = clip;
audioSource.Play();

// 播放一次
audioSource.PlayOneShot(clip);
```

### 淡入播放

```csharp
// 淡入
audioSource.volume = 0;
audioSource.Play();
// 渐变到目标音量
```

### 淡出停止

```csharp
// 渐变降低音量
// 音量到0后停止
audioSource.Stop();
```

### 指定位置播放

```csharp
// 指定位置播放
audioSource.time = position;
audioSource.Play();
```
