XR报错
出现错误信息 "The type name 'XRSettings' could not be found in the namespace 'UnityEngine.XR'" 通常是因为 Unity 的 XR 模块没有被正确启用或者引用的命名空间不正确。从 Unity 2020 版本开始，Unity 的 XR 功能已经从 XR Settings 转移到了 XR Plugin Management 系统 。这意味着你需要在 Unity 的 Package Manager 中启用相应的 XR 插件。
安装 或者删除对应报错脚本
