# DamageTips 伤害提示 - 入门指南

## 包信息

- **对应包**: `cn.etetet.yiuidamagetips`
- **依赖**: Damage Numbers Pro（源插件）、YooAssets

## 系统架构

基于 Damage Numbers Pro 插件，提供伤害数字显示功能。

### 核心概念

1. **DamageType** - 伤害类型
2. **DamageNumber** - 伤害数字
3. **DamageTipsHelper** - 伤害提示帮助类

## 快速开始

### 步骤 1: 安装

1. 安装 Damage Numbers Pro 插件
2. 安装 cn.etetet.yiuidamagetips 包

### 步骤 2: 配置 YooAsset

添加伤害飘字需要的资源

### 步骤 3: 使用 API

```csharp
// 触发伤害飘字
DamageTipsHelper.ShowDamage(target, damage, damageType);
```

## 使用场景

### 场景 1: 玩家攻击

```csharp
public void OnAttack(Entity target, long damage)
{
    DamageTipsHelper.ShowDamage(target, damage, DamageType.Normal);
}
```

### 场景 2: 暴击伤害

```csharp
public void OnCritAttack(Entity target, long damage)
{
    DamageTipsHelper.ShowDamage(target, damage, DamageType.Crit);
}
```

### 场景 3: 治疗效果

```csharp
public void OnHeal(Entity target, long healAmount)
{
    DamageTipsHelper.ShowDamage(target, healAmount, DamageType.Heal);
}
```
