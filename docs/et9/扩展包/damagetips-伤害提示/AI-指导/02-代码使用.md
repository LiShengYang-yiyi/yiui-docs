## 2. 伤害显示

### 基础显示

```csharp
// 显示伤害
DamageTipsHelper.ShowDamage(target, damage);

// 显示治疗
DamageTipsHelper.ShowDamage(target, healAmount, DamageType.Heal);
```

### 指定位置

```csharp
// 指定显示位置
DamageTipsHelper.ShowDamage(target, damage, position);
```

### 带特效

```csharp
// 带特效显示
DamageTipsHelper.ShowDamage(target, damage, effectName);
```

---

## 3. 自定义样式

### 颜色

```csharp
// 设置颜色
DamageTipsHelper.ShowDamage(target, damage, color: Color.red);
```

### 字体大小

```csharp
// 设置字体大小
DamageTipsHelper.ShowDamage(target, damage, fontSize: 36);
```

### 动画

```csharp
// 设置动画类型
DamageTipsHelper.ShowDamage(target, damage, animType: AnimType.Bounce);
```

## 4. 伤害类型

### 普通伤害

```csharp
DamageTipsHelper.ShowDamage(target, damage, DamageType.Normal);
```

### 暴击伤害

```csharp
DamageTipsHelper.ShowDamage(target, critDamage, DamageType.Crit);
```

### 闪避

```csharp
DamageTipsHelper.ShowDamage(target, 0, DamageType.Miss);
```

### 治疗

```csharp
DamageTipsHelper.ShowDamage(target, heal, DamageType.Heal);
```

### 格挡

```csharp
DamageTipsHelper.ShowDamage(target, 0, DamageType.Block);
```

## 5. 性能优化

### 数量限制

```csharp
// 限制同时显示数量
DamageTipsHelper.SetMaxCount(20);
```

### 冷却时间

```csharp
// 设置同目标冷却
DamageTipsHelper.SetTargetCooldown(target, 0.1f);
```

### 对象池

```csharp
// 使用对象池
DamageTipsHelper.UsePool = true;
```
