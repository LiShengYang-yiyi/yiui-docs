# SuperScroll 循环列表 - 入门指南

## 包信息

- **对应包**: `cn.etetet.yiuisuperscroll`
- **测试版**: 功能持续扩展中

## 系统架构

基于 SuperScroll 插件，提供高性能循环列表。

### 核心组件

- **LoopScrollRect** - 循环列表核心组件
- **LoopScrollPrefabSource** - 预制体数据源
- **LoopScrollItem** - 列表项

## 快速开始

### 步骤 1: 安装

1. 安装 cn.etetet.yiuisuperscroll
2. 可选：安装 demo 包学习案例

### 步骤 2: 创建列表

1. 创建 LoopScrollRect
2. 设置预制体
3. 设置数据

### 步骤 3: 使用

```csharp
// 设置数据
scrollRect.totalCount = dataList.Count;
scrollRect.prefabSource = prefabSource;
scrollRect.RefillCells();
```
