 转换符 
 不懂就看这个 
 在C#中格式化字符串 
 
 数值 本地显示 格式化字符串 
 程序中已默认 {0:X} 
 只需要填X的部分 
 没填的默认使用 G 
 
 比如 
 你想保留2位小数  填入: F2 
 你想显示百分比   填入: P0 没有小数 P2 保留2位小数
