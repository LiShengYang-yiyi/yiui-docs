# 数值公式

 
![Image](/images/CB5ewIAQZiBf1Ykp2DDcMj22nHb_0_81cbc665.png)
  本质是把之前的方法改为Invoke 分发 
 
![Image](/images/HmIPwp6emi1nFZk0aMvcJSPtnfe_1_341be4f6.png)
 
 这样不同的数值可以使用不同的公式 
 这样公式不再是统一,可以高度自定义 
 大部分情况下都可以不用这个功能  用默认的公式 0就可以了 (不填就是0) 
 只有真正遇到特殊需求后才会用到, 自己扩展新的公式 
 
 公式ID 在数值定义那里设定
