 5. 一些有用的示例 
 代码块 
 String . Format (”{ 0 :$#,## 0.00 ;($#,## 0.00 ); Zero }”, value ); 
 上面代码如果传入1243.50将输出 “$1,240.00″ 。如果传入负数，则输出相同的格式，不同的是带括号。如果传入0则输出0. 
 代码块 
 String . Format (”{ 0 :(###) ###-####}”, 8005551212 ); 
 上面代码将输出 “(800) 555-1212″.
