# Numeric 数值系统 - 入门指南

## 包信息

- **对应包**: `cn.etetet.yiuinumeric`
- **依赖**: Luban 配置解决方案、cn.etetet.yiuicodeanalysis

## 系统架构

### 核心概念

1. **NumericType** - 数值类型枚举
2. **NumericComponent** - 数值组件
3. **NumericChangeEvent** - 数值变化事件

### 数值类型

- 基础数值：HP, MP, Level, Exp
- 属性数值：Attack, Defense, Speed
- 战斗数值：CriRate, CriDamage, MoveSpeed

## 快速开始

### 步骤 1: 安装

1. 安装 cn.etetet.yiuinumeric.config 包
2. 安装 cn.etetet.yiuicodeanalysis 包

### 步骤 2: 生成配置

编译项目生成 Numeric 配置

### 步骤 3: 使用

```csharp
// 获取数值
var hp = numericComponent.GetValue(NumericType.HP);

// 设置数值
numericComponent.SetValue(NumericType.HP, 100);

// 数值变化
numericComponent.SetValue(NumericType.HP, -10);  // 减少10
```
