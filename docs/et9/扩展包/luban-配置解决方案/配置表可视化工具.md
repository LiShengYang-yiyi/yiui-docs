# 配置表可视化工具

 
对于程序来说没有什么压力也简单 
 分包策略是必须的 有利于程序代码的维护 等等好处 
 但是对于策划 美术 只使用配置来说可能理解上会差很多 
 因此提供了一个可视化工具 可以在工具中预览所有配置 
 不必去每个包中查找等等 
 且增加了 包的 别名 备注 / 配置的 别名 备注 
 此功能也可提升配置表的理解 可读性 提升效率 
 
 ⚽ 依赖YIUI / Odin 
 
 YIUI 宏 需要开启YIUI宏  需要有Odin
