 统一改为  cn.etetet.xx包/Luban/Config 
 前缀统一:cn.etetet.xx包/Luban 
 后缀:Config  表示属于 Luban的Config 分类 
 Config 分类将作为客户端主包分类 
 
 既然是分类那就可以自定义 
 所以你可以自己有cn.etetet.xx包/Luban/AAA 
 那么这个AAA将作为独立luban.conf 使用 
 也就是说可以支持多种配置 
 
 举例: 
 这里看到的就是 
 Luban/Localhost 
 Luban/Release 
 这2个分类也是独立的分类 是服务器的起服配置 
 现在只有这2种起服配置 你想扩展呀第三种的时候也非常简单了 
 只需要做个新的文件夹 就是一个新的分类 
 
![Image](/images/KCKFwTDQ8iwyCfkjuCAcJsmdnYb_2_129409a8.png)

 1个包平均不到1分钟就可以改完 无难度
