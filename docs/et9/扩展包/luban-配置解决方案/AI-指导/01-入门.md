# Luban 配置解决方案 - 入门指南

## 包信息

- **对应包**: `cn.etetet.yiuiluban`
- **独立功能包**: 无外部依赖

## 系统架构

### Luban 核心概念

1. **配置表** - 数据表定义
2. **数据类型** - 字段类型定义
3. **代码生成** - 生成 C# 代码
4. **数据导出** - 导出二进制/JSON 数据

### YIUI 集成

- LubanHelper - 配置加载帮助类
- 配置表分类管理

## 快速开始

### 步骤 1: 安装

从 .Template 文件夹复制 config 包到 Packages

### 步骤 2: 创建配置表

1. 创建 Excel 文件
2. 定义字段和类型
3. 添加测试数据

### 步骤 3: 配置导出

1. 配置 Luban 导出设置
2. 执行导出命令

### 步骤 4: 代码使用

```csharp
// 加载配置表
var configs = LubanHelper.GetTable<T>("table_name");

// 获取配置
var config = configs.Get(id);

// 遍历配置
foreach (var config in configs.GetAll())
{
    // 处理配置
}
```
