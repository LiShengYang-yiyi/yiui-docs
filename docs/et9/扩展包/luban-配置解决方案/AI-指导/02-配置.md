## 配置表创建

### Excel 文件结构

```

## @AutoId true

| Id  | Name  | Type  | Desc   |
|-----|-------|-------|--------|
| int | string| int   | string |
| 角色ID| 名称  | 类型  | 描述   |
```

### 常用数据类型

| 类型 | 说明 | 示例 |
|------|------|------|
| int | 整数 | 1001 |

## 配置表关联

### 主表与子表

```csharp
// 主表
public class CharacterConfig
{
    public int Id;
    public string Name;
    public List<int> Skills;
}

// 子表 - 技能配置
public class SkillConfig
{
    public int Id;
    public string Name;
    public int Damage;
}
```

### 引用查询

```csharp
// 获取角色配置
var character = CharacterConfigCategory.Instance.Get(1);

// 获取角色关联的技能
var skills = character.Skills.Select(id => SkillConfigCategory.Instance.Get(id));
```
