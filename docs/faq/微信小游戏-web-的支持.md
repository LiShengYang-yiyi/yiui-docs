本质
YIUI 本质上就是 Unity UGUI的扩展而已
所以UGUI支持 那就支持
代码
从代码层面 因为已知 没有多线程 资源加载需要异步 等等...
所以某些设计上需要避免此类平台不支持的东西
已知
1.
YIUI的内置无限循环列表组件
(已完成)
原生只有同步加载功能  所以这里需要改为异步加载
(TODO 之后YIUI会修改一个异步的出来)
异步 无限循环列表
2.
YIUI中(通用版) 内置了一个小型的异步锁
这个锁本质上是多线程的 所以需要修改这个异步锁
(TODO 修改异步锁 不使用多线程)
ET版是没有这个问题的  ET自带协程锁
其他
ET的原生版本是不支持
需要ET特供的web版才行 ( web版 找熊猫 )
截止目前ET版本没有什么问题 如果你有web版可直接用
ET9版本
已完全支持不存在上述问题 (当然你得有ET9的web版)
这里只是代表YIUI的ET9版本没有其他问题
