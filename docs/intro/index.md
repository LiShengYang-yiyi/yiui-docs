# 入门前言

 
 YIUI 是一个偏大的UI框架 内容非常多 
 你需要有耐心的学习   视频 + 文档 全部的内容 
 [不要遇到问题就问](/intro/) ( 不要还没自己尝试过解决问题就开始问 ) 
 
 本人并非好为人师 
 建议学习YIUI者有一颗良好的学习心态 
 
 YIUI并不推荐刚入门的新人学习 特别是ET框架的YIUI 
 如果你确定要学习 请付出比别人更多的努力 
 
 YIUI基于Unity UGUI 所以你至少得了解 Unity 了解UGUI相关基础知识 
 
 框架内部有非常多的其他框架/插件 遇到问题时请先找对应YIUI文档 然后是官方文档查询 
 如: 
 ET 框架 
 Luban Yooasset  HybridCLR 
 LoopScrollRect 
 I2Localization 
 等等 你遇到的问题是属于哪方面的总要知道吧 
 
 ❗ 遇到问题 
 问答 
 强烈推荐阅读 [《提问的智慧》](https://github.com/ryanhanwu/How-To-Ask-Questions-The-Smart-Way) 、 [《如何向开源社区提问题》](https://github.com/seajs/seajs/issues/545) 和 [《如何有效地报告 Bug》](http://www.chiark.greenend.org.uk/%7Esgtatham/bugs-cn.html) 、 [《如何向开源项目提交无法解答的问题》](https://zhuanlan.zhihu.com/p/25795393) ，更好的问题更容易获得帮助。 
 
 一些报错日志 非常明显 如果英文不好能不能先 翻译 尝试解决? 
 遇到问题有没有先 查文档 找一找相关说明  常见问题中有没有解答 
 自己有没有尝试过根据问题 先 百度 ? AI ? 
 
 针对性的问题 是不是应该先到对应的 官方文档查询 
 比如 你遇到一个热更问题 理应找热更插件 
 
 分析问题 排除法等等定位问题 
 
 💡 提问 
 提问也是一门学问 
 如果你已经尝试过上述解决方案仍然无法解决问题 尝试群里寻求帮助 
 那么你是否应该提出详细的问题 
 不然如何帮助你解决 
 不要开局一张图 一句话 其他内容全靠群友想 
 你的提问对问题的描述越详细越好  要的是 有效信息 
 如: 
 你使用的YIUI什么版本  因为有 通用版  ET也分 7.2 / 8.1 / 9 
 遇到问题的描述   你有尝试过如何解决 期望达到的预期 等等...
