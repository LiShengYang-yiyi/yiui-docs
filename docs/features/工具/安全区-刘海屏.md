 UnitysafeArea API 进行UI刘海屏自适应

 根据项目需求 定义安全区大小 
 以及动态的变化安全区(没有锁定方向的时候) 
 
 如果项目会发行手游  在有底图的全屏界面都需要考虑刘海屏问题 
 设计背景时 要做大尺寸 
 各UI都应做好对应锚点设置 已达到自适应目的
