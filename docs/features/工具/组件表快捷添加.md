 要求 
 1. 上层必须有CDE表 
 ◦ 必须保证属于同一个预制 不能跨预制 
 2. 自动添加为自动命名 
 ◦ 了解自动命名规则 
 ◦ 选好自动命名设置 
 3. 如果命名冲突会添加失败 
 ◦ 失败原因看日志 或 断点
