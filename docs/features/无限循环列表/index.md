组件

在没有很了解的情况下  创建后什么锚点都不要改 只能改大小 否则也会错
不要问为什么有这些限制 这个插件就是这么设计的 具体请看源码
📌
在没彻底了解的情况下 不要修改任何布局设置 都有可能导致各种问题
建议在最基础的情况下 不修改任何布局 来初始化你的LoopScroll
