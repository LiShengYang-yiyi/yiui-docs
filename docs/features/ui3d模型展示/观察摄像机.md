# 观察摄像机

 
![Image](/images/Bd7Gw6lF6ic46jkxWtQcTt4Qnef_1_62abe52f.png)
 
 如果在同一个显示模式下 有很多模型  他们的大小不一样 那么这个通用摄像机就无法达到需求 
 如果你的所有模型都差不多大 也没有什么特殊需求 那么可以直接用默认摄像机 不传入参数即可 
 
 观察摄像机 
 可以在不同模型下显示不同的样子 
 
 
![Image](/images/Bd7Gw6lF6ic46jkxWtQcTt4Qnef_2_2506e04f.png)
 1. 节点的名称 = 传入时的名称 
 可以有多个 在不同显示时使用不同的 
 2. 节点建议放在根节点最上面  可以减少遍历消耗 
 3. 节点不需要打开 可以选择关闭节点 / Camera的关闭 
 4. 预览里面的内容就是UI上呈现的效果 (预制体不能太小否则这个预览效果会有误差) 
 5. 预览相机的设置 = 渲染时的设置
