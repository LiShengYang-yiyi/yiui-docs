# 自动同步摄像机位置

 
一般很少用 
 比如需要展示特写时配合动画 然后摄像机各种旋转同步 
 就可以用上这个功能 
 当然也有可能无法满足任何需求 
 
 
![Image](/images/VAzZw5Vini4zxXkhOjScO5wQnNb_1_217ea781.png)
 
 
![Image](/images/VAzZw5Vini4zxXkhOjScO5wQnNb_2_96b2ba55.gif)
 
 这个同步的位置是  主摄像机的位置 = 观察摄像机的位置 
 如果你没有传入观察摄像机 那么 = 他自己 
 
 当前你也可以直接控制主摄像机的位置 但是这只能通用 
 观察摄像机的目的就是可以在相同展示下 以不同的方式呈现
